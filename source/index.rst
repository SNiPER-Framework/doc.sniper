Welcome to SNiPER User Manual
==============================================

SNiPER is an open-source general purpose software framework for High Energy Physics data processing. It is dedicated for needs of rare-process search experiments, such as neutrino interactions, dark matter, double beta and other.

For more info please visit the `SNiPER Home at GitHub <https://github.com/SNiPER-Framework/sniper>`_.
